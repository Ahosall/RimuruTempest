# Rimuru Tempest - BOT

Um simples bot, multiuso, para o Discord, usando Discord.js.

## Tecnologias

|Tecnologia|Versão|
|---------:|:-----|
| Node.JS  | v8.0^|
| MongoDB  | v3.x^|

## Instalação

```bash
$ npm install
```

## Ativação

### Produção

```bash
$ npm start # Tá mais ou menos ...
```

### Test

```bash
$ npm test # Em desenvolvimento
```

### Dev

```bash
$ npm run dev # Usavél ...
```
